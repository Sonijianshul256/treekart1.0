# treekart-updated-final

Treekart is a React, Firebase, Google Maps, Stripe, and OneSignal web app for renting organic Papaya and Mango trees in rural Rajasthan.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Firebase Functions

```bash
cd functions
npm install
npm run build
```

Copy `.env.example` and `functions/.env.example` before deploying with real Firebase, Stripe, Google Maps, SMTP, and OneSignal credentials.
